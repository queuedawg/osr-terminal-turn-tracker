# What is this tool?
This is a simple, free utility that runs in your terminal to track time in Old School RPGs. I wrote this for my own game, but thought it might be helpful for others, too.

To advance a turn, all you need to do is press a single button. The app automatically tracks torches, lanterns, cleric light spells, magic user light spells, potions of light, prompts wandering monster checks, and prompts you that the party must rest. There are configurable options for almost everything listed above.
